# IPScan Pro

Advanced TCP Port Scanner (Python CLI)

สแกนพอร์ตแบบหลายเธรด พร้อมตรวจจับ Service และ Version เบื้องต้น\
รองรับ Apache / nginx / OpenSSH detection\
แสดงผลแบบมีสีใน Terminal

------------------------------------------------------------------------

## Features

-   ตรวจสอบพอร์ตแบบ TCP connect
-   รองรับสแกนพอร์ตเดียว
-   สแกนช่วงพอร์ตแบบกำหนดเอง
-   โหมด Auto (1--10000)
-   ตรวจจับ Service:
    -   HTTP
    -   HTTPS
    -   SSH
    -   FTP
    -   MySQL
    -   RDP
-   ตรวจจับเวอร์ชัน:
    -   Apache/x.x.x
    -   nginx/x.x.x
    -   OpenSSH_x.x
-   ไม่แสดง Unknown
-   แสดงสีใน Terminal
-   ใช้ได้บน Windows / Linux / macOS
-   ไม่มี dependency ภายนอก (ใช้แค่ Python standard library)

------------------------------------------------------------------------

## Requirements

-   Python 3.7+
-   ไม่ต้องติดตั้ง library เพิ่ม

ตรวจสอบเวอร์ชัน Python:

    python3 --version

------------------------------------------------------------------------

## Installation

ดาวน์โหลดไฟล์:

    ipscan.py

ให้สิทธิ์รัน (Linux/macOS):

    chmod +x ipscan.py

------------------------------------------------------------------------

## Usage

ถ้ารันโดยไม่ใส่ argument:

    python3 ipscan.py

จะแสดง Help และตัวอย่างการใช้งาน

------------------------------------------------------------------------

### 1) Auto Scan (1--10000)

    python3 ipscan.py -au -u 49.228.131.69

------------------------------------------------------------------------

### 2) สแกนพอร์ตเดียว

    python3 ipscan.py -p 22 -u 49.228.131.69

------------------------------------------------------------------------

### 3) สแกนตั้งแต่ 1 ถึง N

    python3 ipscan.py -a 5000 -u 192.168.1.1

------------------------------------------------------------------------

## Output Example

    [+] Target: 49.228.131.69
    [+] Scanning: 1 - 10000
    --------------------------------------------------
    [OPEN] 22  --->  SSH  (OpenSSH_8.4p1)
    [OPEN] 80  --->  HTTP (Apache/2.4.54)
    [OPEN] 443 --->  HTTP (nginx/1.22.1)
    --------------------------------------------------
    [+] Done. Found: 3 services

สีที่ใช้: - เขียว = OPEN - น้ำเงิน = Service - เหลือง = Version

------------------------------------------------------------------------

## Options

  Option   Description
  -------- --------------------
  -u       Target IP address
  -p       Scan single port
  -a       Scan ports 1--N
  -au      Auto scan 1--10000

------------------------------------------------------------------------

## Notes

-   ใช้สแกนเฉพาะระบบที่คุณมีสิทธิ์หรือได้รับอนุญาตเท่านั้น
-   บางระบบอาจปิด banner ทำให้ไม่สามารถดึงเวอร์ชันได้
-   HTTPS อาจไม่แสดงเวอร์ชันเพราะใช้ TLS
